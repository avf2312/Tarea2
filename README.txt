Primeramente antes de entregar datos de como se corre el código se tomaron unas suposiciones debido a la rareza de marcar y desmarcar correos. Tomamos marcar correo favorito como marcar favorito el correo de un usuario, al igual que desmarcar correo favorito que se basa en la misma lógica.

Para obtener la carpeta node_modules necesaria, se necesita hacer npm install

Para instalar las dependencias, se necesita hacer pip install -r requirements.txt

Se sigue con  migrar las tablas hacia la base de datos, esto haciendo bun prisma generate and bun prisma migrate dev correspondientemente. 

Para correr el código de python se necesita correr en una terminal _bun run dev_ (ejecutará "bun run --watch index.ts", es solo una forma más rápida y se encuentra en package.json) para empezar a que el servidor funcione antes de empezar a ejecutar el código de Python.

Para ejecutar el código de python se puede hacer directo de la terminal de VS Code apretando la flechita de empezar o como gusten ejecutar un programa de Python, funcionará de las dos maneras.
